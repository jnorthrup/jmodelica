Bouncing Ball

This FMU encorporates a simple bouncing ball example. Calculated is the movement of a ball from an initial position and speed under influence of gravity. Gravity accelerates the ball towards the ground.

Upon contact with the ground the ball bounces back with reduced speed. This is encorporated by an event that ist triggered initializing the speed with altered sign and value. The physical equations on which this example is based are the following:

Variables Description Start Value Unit
position Postion of the ball 1.0 m
der_position Derivative of ball position 4.0 m/s
Outputs Description Start Value Unit
speed Speed of the ball 4.0 m/s
Parameters Description Start Value Unit
gravity gravity constant -9.81 m/s2
bounce_coef Bouncing coefficient of ball 0.5 -
